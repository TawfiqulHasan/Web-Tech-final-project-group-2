# Warehouse Admin Panel

A PHP + MySQL warehouse admin panel following MVC architecture.

## Main Features

- Login and logout
- Session authentication
- Role-based access control
- Dashboard
- Product management
- Category management
- Warehouse management
- Supplier management
- Purchase order management
- Stock transaction management
- Low stock report
- Activity log
- Basic PHP validation
- Basic JavaScript validation
- Ajax/JSON low stock feature
- GitHub teamwork guide
- Report template
